My Impactful Projects

New Store Location Strategy

Developed a comprehensive data model to identify optimal locations for new 802Spirits stores. The analysis combined census data, existing sales data, flood plain data, and AOT traffic density information to pinpoint high-potential sites.

  • **Impact:** Generated a projected **$1.87 million in new revenue for FY25** by providing data-backed recommendations for store locations.
  • **Tools:** Python, GIS tools, advanced data analysis techniques, data modeling.

Compliance & Public Health - Synar Sampling Strategy

Designed a statistically valid random sampling process for tobacco compliance checks across three geographic zones, reducing annual checks from 1271 to 324. This action provided a repeatable methodology that maintains federal compliance while optimizing enforcement resources.

  • **Impact:** Provided a repeatable sampling methodology that maintains compliance with federal Synar requirements and optimizes enforcement resources.
  • **Tools:** Power BI, Power Query (M), random sampling algorithms, Salesforce integration planning.

Enterprise KPIs & Performance Strategy

Initiated a unified KPI framework by engaging managers across multiple divisions (Finance, Licensing, Enforcement, etc.) to define productivity and identify data sources. This established the foundation for cross-departmental dashboards that reflect real operational impact.

  • **Impact:** Set the foundation for cross-departmental performance dashboards that reflect real operational impact, not vanity metrics.
  • **Tools:** Facilitation, stakeholder engagement, KPI development, strategic alignment.

LiquorCoordinator Route Optimization

Developed a route optimization solution using Python integrations, k-means clustering, and Haversine distance calculations. This project improved coordinator efficiency and agency satisfaction by fitting site visits into viable clusters for an eight-hour workday.

  • **Impact:** Improved coordinator efficiency and improved agency satisfaction.
  • **Tools:** Python, k-means clustering, Haversine, Power BI.

Outlook Email Migration to SharePoint

Automated the migration of Outlook group emails to SharePoint lists using Power Automate. This provided real-time operational visibility into average response times, first response times, and email categorization, including sentiment analysis.

  • **Impact:** Provided operational visibility, including average response times, first response time categorization of emails.
  • **Tools:** Power Automate, SharePoint, Power BI.
